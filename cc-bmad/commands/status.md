# /bmad:status

> 현재 BMAD 워크플로우의 진행 상태를 확인합니다.

---

## Triggers

- 워크플로우 진행 상황 확인 시
- 어떤 페이즈가 진행 중인지 확인 시
- 블로커나 실패 상태 확인 시

## 사용법

### 기본 사용

```bash
# 전체 상태 확인
/bmad:status
```

### 특정 페이즈 상태

```bash
# 특정 페이즈 상세 확인
/bmad:status --phase solutioning
```

### 특정 이슈 상태

```bash
# 이슈 번호로 상태 확인
/bmad:status --issue 1810
```

---

## 옵션

| 옵션 | 기본값 | 설명 |
|------|--------|------|
| `--phase` | 전체 | 특정 페이즈만 확인 |
| `--issue` | 현재 | 특정 이슈 상태 확인 |
| `--verbose` | false | 상세 정보 표시 |

## 상태 코드

| 상태 | 아이콘 | 설명 |
|------|--------|------|
| PENDING | ⏳ | 대기 중 |
| IN_PROGRESS | 🔄 | 진행 중 |
| PASSED | ✅ | 통과 |
| FAILED | ❌ | 실패 |
| SKIPPED | ⏭️ | 스킵됨 |
| WARNING | ⚠️ | 경고 (통과됨) |
