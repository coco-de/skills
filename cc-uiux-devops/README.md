# cc-uiux-devops

CoCode 팀을 위한 DevOps 플러그인입니다.

## 개요

Flutter Web/App 빌드 및 Serverpod 서버 배포를 위한 CI/CD 파이프라인, 배포 전략, 모니터링을 지원합니다.

## 스킬 목록

### cicd-pipeline
- GitHub Actions 기반 Flutter 빌드 파이프라인
- Serverpod 배포 자동화

### deployment
- Flutter Web/App 배포 전략
- Serverpod 서버 배포 관리

### monitoring
- 앱 크래시 리포팅
- 서버 헬스 체크
- 로그 집계 및 분석

## 기술 스택

- **CI/CD**: GitHub Actions
- **빌드**: Flutter Build, Dart Compile
- **배포**: Flutter Web, App Store, Google Play, Serverpod
- **모니터링**: Crashlytics, Sentry, 서버 로그
