# cc-uiux-backend

CoCode 팀을 위한 백엔드 개발 플러그인입니다.

## 개요

Serverpod 전용 백엔드 개발을 지원합니다. API 설계, 데이터베이스 모델링, 직렬화, 인증 등 Serverpod 프레임워크의 핵심 기능을 다룹니다.

## 스킬 목록

### api-design
- Serverpod 엔드포인트 설계
- 직렬화(Serialization) 패턴
- 인증(Authentication) 구현

### database-design
- Serverpod 모델 정의
- 테이블 관계(Relations) 설정
- 마이그레이션(Migrations) 관리

## 기술 스택

- **프레임워크**: Serverpod
- **데이터베이스**: PostgreSQL (Serverpod 기본)
- **언어**: Dart
- **아키텍처**: Clean Architecture
