# Cohort Analysis (코호트 분석)

## 트리거
- 사용자 리텐션 또는 이탈 패턴을 분석해야 할 때
- "코호트", "리텐션", "cohort", "retention" 키워드가 언급될 때
- 특정 사용자 그룹의 시간별 행동 변화를 추적해야 할 때
- 기능 출시 전후 사용자 행동 비교가 필요할 때

## 동작
1. **코호트 정의**:
   - 시간 기반 코호트 (가입 주/월별)
   - 행동 기반 코호트 (특정 액션 수행 여부)
   - 속성 기반 코호트 (사용자 세그먼트, 획득 채널)
   - 코호트 크기 및 유의미성 검증
2. **리텐션 분석**:
   - N-day 리텐션 (Day 1, Day 7, Day 30)
   - 주간/월간 리텐션 커브
   - 롤링 리텐션 vs 클래식 리텐션 비교
   - 리텐션 벤치마크 대비 평가
3. **인게이지먼트 메트릭 분석**:
   - 세션 빈도 및 시간
   - 핵심 기능 사용률 (코호트별)
   - 파워 유저 비율 추적
   - Stickiness (DAU/MAU) 분석
4. **코호트 비교 분석**:
   - 코호트 간 리텐션 커브 비교
   - 개선/악화 트렌드 식별
   - 기능 출시 전후 코호트 비교
   - 통계적 유의성 검증
5. **인사이트 도출 및 액션**:
   - 이탈 원인 가설 수립
   - 리텐션 개선 레버 식별
   - Aha Moment 분석 (활성화 기준 도출)

## 출력
- 리텐션 테이블 (코호트 x 기간 매트릭스)
- 리텐션 커브 데이터
- 인게이지먼트 메트릭 대시보드 설계
- 코호트 비교 분석 보고서
- 리텐션 개선 액션 아이템 목록

## 참고
- Flutter 앱과 Jaspr Web 사용자를 별도 코호트로 분석하여 플랫폼별 리텐션 차이 파악
- Serverpod의 세션 및 인증 데이터를 활용한 코호트 구성
- 크로스 플랫폼 사용자 (모바일 + 웹 동시 사용)의 리텐션 특성 분석
- ClickHouse 쿼리로 대규모 코호트 분석 수행 시 파티션 및 샘플링 전략 활용
