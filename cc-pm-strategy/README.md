# cc-pm-strategy

제품 전략(Product Strategy) 스킬 플러그인입니다.

## 개요

제품 비전 수립, 비즈니스 모델 분석, 경쟁 분석 등 제품 전략 단계에서 필요한 PM 스킬을 제공합니다. BMAD 방법론과 중복되지 않는 전략적 분석 및 의사결정 기법에 집중합니다.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| [product-vision](skills/product-vision/SKILL.md) | 제품 비전 수립 - 비전 선언문, 노스스타 메트릭 |
| [business-model](skills/business-model/SKILL.md) | 비즈니스 모델 분석 - 수익 모델, 유닛 이코노믹스 |
| [competitive-analysis](skills/competitive-analysis/SKILL.md) | 경쟁 분석 - 시장 매핑, 차별화 전략 |

## 대상 팀

CoCode 팀 (Dart 풀스택: Flutter + Jaspr Web + Serverpod)

## 설치

Claude Code 플러그인으로 설치하여 사용합니다.
