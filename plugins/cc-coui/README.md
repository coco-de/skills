# cc-coui

CoUI 크로스플랫폼 컴포넌트 라이브러리 스킬 (Flutter + Jaspr Web)

## 컴포넌트 (26개)

| 카테고리 | 컴포넌트 |
|---------|---------|
| **입력** | button, checkbox, date-picker, form, input, select, slider, toggle |
| **표시** | avatar, badge, card, progress, table, text |
| **네비게이션** | menu, navigation, tabs |
| **오버레이** | dialog, drawer, popover, toast, tooltip |
| **레이아웃** | accordion, calendar |
| **테마** | theme |

## 사용법

각 컴포넌트의 `SKILL.md`에 Flutter 및 Jaspr Web 구현 가이드가 포함되어 있습니다.
