# cc-uiux-accessibility

Cocode 팀을 위한 접근성 플러그인입니다.

## 개요

Flutter 모바일/데스크톱 앱과 Jaspr Web 환경에서의 접근성(a11y) 구현을 지원합니다. 스크린 리더, 키보드 네비게이션, WCAG 2.1 준수 등을 포함합니다.

## 스킬 목록

### flutter-a11y
- Flutter Semantics 위젯 활용
- 스크린 리더 지원 (TalkBack, VoiceOver)
- 명암비(contrast ratio) 검증

### web-a11y
- Jaspr Web ARIA 레이블 적용
- 키보드 네비게이션 구현
- WCAG 2.1 준수 검증

## 기술 스택

- **모바일/데스크톱**: Flutter Semantics API
- **웹**: Jaspr Web, ARIA 표준
- **표준**: WCAG 2.1 AA/AAA
- **컴포넌트**: CoUI 접근성 지원
