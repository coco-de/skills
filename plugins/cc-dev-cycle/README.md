# cc-dev-cycle

개발 사이클 자동화 플러그인 (Issue Cycle, Bug Cycle, Session, ZenHub)

## Skills

- `workflow` - 이슈 기반 개발 사이클 관리
- `session` - 세션 학습/규칙 추출

## Commands

- `workflow/*` - 개발 사이클 커맨드 (issue-cycle, bug-cycle, issue-processor)
- `session/*` - 세션 관련 커맨드
- `zenhub/*` - ZenHub 연동 커맨드
- `openapi/*` - OpenAPI 관련 커맨드

## Agents

- `sequential-workflow` - Epic 하위 이슈 순차 처리 에이전트
- `mcp-debug` - MCP 디버그 에이전트
