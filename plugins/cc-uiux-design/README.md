# cc-uiux-design

CoCode 팀을 위한 UI/UX 디자인 플러그인입니다.

## 개요

Dart 풀스택 환경(Flutter + Jaspr Web + Serverpod)에서 CoUI 컴포넌트 라이브러리와 DaisyUI 테마를 활용한 디자인 시스템 관리 및 디자인 리뷰를 지원합니다.

## 스킬 목록

### design-system
- CoUI 디자인 토큰 관리 (색상, 타이포그래피, 간격)
- DaisyUI 테마 커스터마이징
- 컴포넌트 테마 설정

### design-review
- CoUI 표준 기반 UI 일관성 검토
- 디자인 토큰 준수 여부 확인

## 기술 스택

- **컴포넌트**: CoUI (자체 컴포넌트 라이브러리)
- **테마**: DaisyUI 기반 테마 시스템
- **프레임워크**: Flutter, Jaspr Web
- **패턴**: BLoC 패턴, Clean Architecture
