# cc-backend

Serverpod 백엔드 심화 플러그인 — ORM, Auth, Caching, Testing, API Design, Architecture

> `cc-serverpod`가 모델/엔드포인트/마이그레이션 생성을 담당한다면, `cc-backend`는 심화 패턴과 아키텍처를 다룹니다.

## Skills

| 스킬 | 설명 |
|------|------|
| `database` | Serverpod ORM 심화 (필터, 관계, 트랜잭션, 잠금, 마이그레이션) |
| `auth` | 인증 모듈 (IDP, JWT, 서버사이드 세션, Flutter UI) |
| `caching` | 캐싱 (Local, Redis, CacheMissHandler) |
| `testing` | 테스팅 심화 (withServerpod, TDD, 롤백, 스트림) |
| `logging` | 로깅 & 모니터링 (session.log, 보존, Insights) |
| `api-design` | API 설계 원칙 (REST, 버전관리, 에러 처리, 페이지네이션) |
| `architecture` | 아키텍처 패턴 (Clean Architecture, DDD for Serverpod) |

## Agents

| 에이전트 | 설명 |
|---------|------|
| `backend-architect` | 백엔드 아키텍처 설계 전문가 |
| `tdd-orchestrator` | TDD Red-Green-Refactor 오케스트레이터 |

## Rules

| 규칙 | 설명 |
|------|------|
| `backend-conventions` | 백엔드 코딩 컨벤션 |

## cc-serverpod와의 관계

- **cc-serverpod**: 모델(.spy.yaml) 생성, 엔드포인트 생성, 마이그레이션 병합, 예외 패턴
- **cc-backend**: ORM 심화 쿼리, 인증 설정, 캐싱 전략, TDD, API 설계, 아키텍처
