# cc-pm-gtm

Go-to-Market 스킬 플러그인입니다.

## 개요

ICP(Ideal Customer Profile) 정의, 메시징 프레임워크, GTM 모션 계획 등 시장 진입 전략에 필요한 PM 스킬을 제공합니다. BMAD 방법론과 중복되지 않는 시장 진입 및 성장 전략 기법에 집중합니다.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| [icp-definition](skills/icp-definition/SKILL.md) | ICP 정의 - 이상적 고객 프로필 수립 |
| [messaging-framework](skills/messaging-framework/SKILL.md) | 제품 메시징 및 포지셔닝 프레임워크 |
| [gtm-motion](skills/gtm-motion/SKILL.md) | GTM 모션 계획 - 런칭 전략, 채널 선정 |

## 대상 팀

Cocode 팀 (Dart 풀스택: Flutter + Jaspr Web + Serverpod)

## 설치

Claude Code 플러그인으로 설치하여 사용합니다.
