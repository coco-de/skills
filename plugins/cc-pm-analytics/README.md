# cc-pm-analytics

데이터 분석(Product Analytics) 스킬 플러그인입니다.

## 개요

SQL 기반 분석, 코호트 분석, A/B 테스트 등 데이터 기반 의사결정에 필요한 PM 스킬을 제공합니다. BMAD 방법론과 중복되지 않는 정량적 분석 기법에 집중합니다.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| [sql-analysis](skills/sql-analysis/SKILL.md) | SQL 기반 제품 분석 - 쿼리, 대시보드 (ClickHouse 호환) |
| [cohort-analysis](skills/cohort-analysis/SKILL.md) | 코호트 분석 - 리텐션, 인게이지먼트 메트릭 |
| [ab-testing](skills/ab-testing/SKILL.md) | A/B 테스트 - 실험 설정, 통계적 유의성, 결과 해석 |

## 대상 팀

CoCode 팀 (Dart 풀스택: Flutter + Jaspr Web + Serverpod)

## 설치

Claude Code 플러그인으로 설치하여 사용합니다.
