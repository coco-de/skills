# cc-flutter-dev

Flutter 개발 핵심 플러그인 (UI, Feature, BLoC, Test)

## Skills

- `workflow` - 개발 워크플로우 스킬
- `flutter-ui` - Flutter UI 개발 스킬

## Agents

- `bloc` - BLoC 패턴 에이전트
- `feature` - Feature 개발 에이전트
- `flutter-ui` - Flutter UI 에이전트
- `flutter-image-optimizer` - 이미지 최적화 에이전트
- `test` - 테스트 에이전트
- `console-feature` - 콘솔 Feature 에이전트

## Commands

- `bloc` - BLoC 생성 커맨드
- `test` - 테스트 실행 커맨드
- `flutter-ui` - Flutter UI 커맨드
- `feature/*` - Feature 관련 커맨드
- `bdd/*` - BDD 테스트 커맨드
- `figma/*` - Figma 연동 커맨드
