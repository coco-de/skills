# cc-pm-discovery

제품 발견(Product Discovery) 스킬 플러그인입니다.

## 개요

사용자 인터뷰, 실험 설계, 가정 테스트 등 제품 발견 단계에서 필요한 PM 스킬을 제공합니다. BMAD 방법론과 중복되지 않는 실무 중심의 발견 기법에 집중합니다.

## 스킬 목록

| 스킬 | 설명 |
|------|------|
| [user-interview](skills/user-interview/SKILL.md) | 사용자 인터뷰 실행 - 질문 프레임워크, 인터뷰 기법, 인사이트 종합 |
| [experiment-design](skills/experiment-design/SKILL.md) | 실험 설계 - 가설 수립, 테스트 설계, 결과 분석 |
| [assumption-testing](skills/assumption-testing/SKILL.md) | 가정 테스트 - 가장 위험한 가정 식별, 검증 방법 |

## 대상 팀

CoCode 팀 (Dart 풀스택: Flutter + Jaspr Web + Serverpod)

## 설치

Claude Code 플러그인으로 설치하여 사용합니다.
