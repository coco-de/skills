# cc-workflow

개발 워크플로우 플러그인 (Issue Cycle, Bug Cycle, Session, ZenHub)

## Skills

- `workflow` - 워크플로우 관리 스킬
- `session` - 세션 관리 스킬

## Commands

- `workflow/*` - 워크플로우 관련 커맨드
- `session/*` - 세션 관련 커맨드
- `zenhub/*` - ZenHub 연동 커맨드
- `openapi/*` - OpenAPI 관련 커맨드

## Agents

- `sequential-workflow` - 순차 워크플로우 에이전트
- `mcp-debug` - MCP 디버그 에이전트
