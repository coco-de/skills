# cc-uiux-testing

Cocode 팀을 위한 테스팅 플러그인입니다.

## 개요

Dart/Flutter 테스트 프레임워크를 활용한 단위 테스트, 위젯 테스트, 통합 테스트, 성능 테스트를 지원합니다. BLoC 테스트와 CoUI 컴포넌트 테스트를 포함합니다.

## 스킬 목록

### unit-testing
- Dart 단위 테스트 구조
- Mockito/Mocktail 기반 모킹
- BLoC 테스트 패턴

### widget-testing
- Flutter 위젯 테스트
- Golden 테스트 (스크린샷 비교)
- CoUI 컴포넌트 테스트

### integration-testing
- End-to-End 테스트
- Serverpod 통합 테스트

### performance-testing
- Flutter DevTools 프로파일링
- 프레임 분석 및 최적화

## 기술 스택

- **테스트 프레임워크**: flutter_test, dart test
- **모킹**: Mockito, Mocktail
- **BLoC 테스트**: bloc_test
- **성능**: Flutter DevTools
