# Product Management

4개의 PM 플러그인이 제품 라이프사이클 전체를 커버합니다.

## cc-pm-discovery

**제품 발견** — 사용자 리서치와 실험 설계를 체계화합니다.

| 항목 | 내용 |
|------|------|
| Skills | 3개 |
| 기능 | 사용자 인터뷰, 실험 설계, 가정 테스트 |

### 설치

```bash
claude plugins install coco-de/skills/plugins/cc-pm-discovery
```
