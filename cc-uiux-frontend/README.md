# cc-uiux-frontend

CoCode 팀을 위한 프론트엔드 개발 플러그인입니다.

## 개요

CoUI 컴포넌트 라이브러리를 활용한 Flutter/Dart 전용 프론트엔드 개발 패턴을 지원합니다. BLoC 패턴과 Clean Architecture 기반의 구조화된 개발을 안내합니다.

## 스킬 목록

### flutter-patterns
- BLoC 패턴 기반 상태 관리
- Clean Architecture 레이어 분리
- CoUI 컴포넌트 통합 패턴

### jaspr-patterns
- Jaspr Web SSR(서버 사이드 렌더링)
- 라우팅 설정 및 관리
- CoUI 웹 컴포넌트 활용

### responsive-design
- 반응형/적응형 디자인 전략
- 브레이크포인트 설정
- 플랫폼 감지 및 레이아웃 전략

## 기술 스택

- **프레임워크**: Flutter, Jaspr Web
- **상태 관리**: BLoC 패턴
- **아키텍처**: Clean Architecture
- **컴포넌트**: CoUI
- **언어**: Dart
